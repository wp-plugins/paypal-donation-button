== Description ==

A simple PayPal donation button WordPress plugin.

= Usage =

After installing and Activating the plugin, go to WordPress Admin Dahsbord and then go to

`Settings >> PayPal Donation`

Here you can change default settings. By default your WordPress site admin email will be used as PayPal ID. So if you have different PayPal ID, change it here. Also choose donate button size and select your desired currency.

After saving PayPal Donation Settings, just paste the following shortcode at any page or text widget where your want to show "PayPal Donation Button"

`[paypal_donation_button]`

== Installation ==
Installing the plugins is just like installing other WordPress plugins. If you don't know how to install plugins, please review the two options below:

Install by Search

* From your WordPress dashboard, choose 'Add New' under the 'Plugins' category.
* Search for 'PayPal Donation Button' a plugin will come called 'PayPal Donation Button' and Click 'Install Now' and confirm your installation by clicking 'ok'
* The plugin will download and install. Just click 'Activate Plugin' to activate it.

Install by ZIP File

* From your WordPress dashboard, choose 'Add New' under the 'Plugins' category.
* Select 'Upload' from the set of links at the top of the page (the second link)
* From here, browse for the zip file included in your plugin titled 'paypal-donation-button.zip' and click the 'Install Now' button
* Once installation is complete, activate the plugin to enable its features.

Install by FTP

* Find the directory titles 'paypal-donation-button' and upload it and all files within to the plugins directory of your WordPress install (WORDPRESS-DIRECTORY/wp-content/plugins/) [e.g. www.yourdomain.com/wp-content/plugins/]
* From your WordPress dashboard, choose 'Installed Plugins' option under the 'Plugins' category
* Locate the newly added plugin and click on the \'Activate\' link to enable its features.
